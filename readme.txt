Simply adds a Home or Blog label to a page title if it's manually set to display as the home or blog page.

== Description ==

Simply adds a label next to a page when you set it to display as the home page or a blog page. Great for websites with a long list of pages!

== Installation ==

Download the zip file and upload to your WordPress installation. Plugin automatically finds and adds relevant labels.
